## ✨ Overview

**F-Closer** is a state-of-the-art, glassmorphism-driven relationship management platform engineered for people who believe that meaningful connections deserve intentional care. Built with **Next.js 16**, **React 19**, and **Tailwind CSS v4**, the platform introduces the concept of *Relationship Intentionality* — a curated workspace for monitoring, nurturing, and never losing track of the bonds that define your life.

Every friend gets a **Connection Goal** — a personal cadence target (e.g., "talk every 7 days"). The app's pulse engine continuously compares the last interaction date against your cadence goal, surfacing visual alerts for friendships that need attention. Backed by a **Recharts analytics layer**, a **Quick Check-In logging system**, and a **Living Timeline** of shared moments, F-Closer transforms the abstract idea of "staying in touch" into measurable, beautiful action.

## ❌ The Problem & ✅ The Solution

> **Meaningful friendships fade in the noise of daily life.**

We rely on passive social media updates instead of reaching out. High-value relationships slowly become digital ghosts — not because we stopped caring, but because life got busy and no tool helped us stay accountable.

| ❌ The Problem | ✅ F-Closer's Solution |
| ------------------------------------------------------------ | --------------------------------------------------------------------------------- |
| No visibility into which friendships are drifting | **Connection Pulse** engine — visually surfaces "Needs Attention" vs "On Track" |
| No way to set personal connection frequency goals | **Custom Connection Cadence** per friend (daily, weekly, monthly targets) |
| Interaction history scattered across apps & memory | **Living Timeline** — a chronological log of calls, texts, and meetups |
| No data on how you actually spend social time | **Friendship Analytics** — Recharts visualizations of interaction habits |
| Generic dashboards with no emotional resonance | **Cinematic glassmorphic UI** — each friend card is an immersive relationship portal |
| Hard to remember context before reaching out | **Friend Detail Page** — full relationship history visible at one glance |

## 🚀 Key Features

- **💜 The Circle (Home)** — A cinematic dashboard of your inner circle. Each friend card shows who is "On Track" and who "Needs Attention" based on their connection cadence target.
- **🔍 Deep Connection (Friend Details)** — Set custom connection cadences per friend and use the **Quick Check-In** system to log moments in real-time.
- **📅 Living History (Timeline)** — A chronological tapestry of your social investment — every call, text, and meetup logged and filterable.
- **📊 Friendship Analytics (Stats)** — **Recharts** visualizations revealing how you prefer to reach out and where your most valuable social time goes.
- **🔔 Smart Pulse Alerts** — Real-time comparison of last interaction vs. cadence goal, surfacing overdue connections automatically.
- **🎬 Motion Animations** — Fluid entrance animations powered by **Motion (Framer Motion)** for a premium, alive interface.

## 🛠️ Installation & Setup

1. **Clone & Install**

   ```bash
   git clone https://github.com/CoderGUY47/f-closer.git
   cd f-closer
   npm install
   ```

2. **Develop Locally**

   ```bash
   npm run dev
   ```

   Access the web app at `http://localhost:3000`.

3. **Production Build**

   ```bash
   npm run build
   ```

## 🚢 Production Deployment

F-Closer is pre-configured for **Vercel** deployment out of the box.

- **Vercel Build Command:** `npm run build` (runs `next build`)
- **Vercel Output Directory:** `.next`
- **Framework Preset:** Next.js (auto-detected by Vercel)
